##  cheat sheet

### Launch sass watcher
```bash
sass sass:.tmp --watch
```
### Launch tailwind watcher
```bash
npx tailwindcss -i ./.tmp/main.css -o ./public/css/style.css --watch
```
### Nodemon server
```bash
nodemon index.js
```
### Start Mongodb.service
```bash
sudo systemctl start mongodb.service
```
### Stop Mongodb.service
```bash
sudo systemctl stop mongodb.service
```
